# Money Tracker

本專案為一個網頁版個人財務管理應用程式，旨在協助使用者記錄日常收支、管理消費類別，並透過數據視覺化介面分析財務狀況。

## 線上預覽 (Live Demo)

[https://dynamix3310.vercel.app/](https://dynamix3310.vercel.app/)

## 功能說明

本系統包含以下核心功能：

* **收支記錄管理**：支援新增、修改及刪除收入與支出項目。
* **財務總覽**：即時計算並顯示當前總收入、總支出及帳戶結餘。
* **分類系統**：提供多種消費類別標籤，便於帳務歸納與檢索。
* **數據分析**：整合圖表功能，視覺化呈現各類別消費佔比與資金流向。
* **響應式介面**：介面設計適配桌面與行動裝置瀏覽器。

## 開發環境建置

若需在本地端運行此專案，請參照以下步驟：

### 1. 環境需求
請確保開發環境已安裝 Node.js 與 npm (或 yarn)。

### 2. 下載專案
```bash
git clone [https://github.com/Dynamix3310/money_tracker.git](https://github.com/Dynamix3310/money_tracker.git)
````

### 3\. 安裝相依套件

進入專案目錄並安裝所需套件：

```bash
cd money_tracker
npm install
```

### 4\. 啟動開發伺服器

執行以下指令以啟動本地開發環境：

```bash
npm start
# 若使用 Vite 或其他構建工具，可能為：
# npm run dev
```

啟動後，請於瀏覽器開啟 `http://localhost:3000` (或終端機顯示之連接埠) 進行瀏覽。
